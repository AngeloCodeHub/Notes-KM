# 書籤的管理邏輯

書籤當成雲端硬碟要能搜尋（Search）、排序（Sort）、過濾（Filter）、分類（Category）、標籤（Tab）

# 工作流準則（憲章）

1. 工作瀏覽器有一個準備匯出到 Raindrop
2. 當前瀏覽器分頁作為*當前工作*空間，非當前工作不要留並存成書籤
3. 瀏覽器工作空間不同主題應該分新視窗或是 Group 以利快速存成書籤
4. 書籤作為筆記其中一部分
5. 存成書籤要明確定義並可搜尋
6. 日期書籤存留必須在三個工作日處理掉

## Raindrop Tip

- 搜尋會套用子收藏夾
- SideBar 顯示的書籤數量不包含子收藏夾的書籤數量
- 排序按照名稱是以 title，按照網站預設去掉 "www." 之後去排序
