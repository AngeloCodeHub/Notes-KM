是的，GitHub Copilot 支持在訂閱期間更換方案。  
如果你目前是使用 Pro 方案，但還沒到一個月就想升級到 Pro+ 方案，你可以隨時在 GitHub 的 Copilot 訂閱管理頁面進行升級。一般會採用「按比例計算」的方式，未使用的 Pro 方案費用會自動抵扣到 Pro+ 的新訂閱，並立即生效。

**操作方式：**
1. 前往 [GitHub Copilot 訂閱頁面](https://github.com/settings/copilot)。
2. 點選「更換方案」或「升級」按鈕。
3. 選擇 Pro+ 方案並確認升級。
4. GitHub 會根據你已使用的時間自動計算費用。

**注意事項：**
- 升級方案後，權益會立即生效。
- 已付費但尚未使用完的部分會自動按比例折算。
- 具體計費細節可參考 GitHub Copilot 的官方說明或帳單記錄。

如果你是 **Yearly (年繳)** 訂閱 GitHub Copilot，升級方案時也會進行按比例折算，但細節略有不同：

- 當你從 Pro 年繳升級到 Pro+ 年繳，GitHub 會根據你剩餘的 Pro 訂閱時間，按比例將未使用的金額折抵到 Pro+，你只需補足差額。
- 若你升級到 Pro+ 月繳，剩餘的 Pro 年繳費用也會按比例抵扣到新方案（月繳或年繳）。

**官方計費說明：**
> 升級後，未使用的訂閱金額會自動折抵到新方案，並立即生效。

**注意：**
- 年繳方案的折算通常會更加細緻，精確到剩餘天數。
- 如果是從年繳降級到月繳或取消，GitHub 可能會以帳戶餘額或退款的方式處理。

如需查詢自己帳單的折算明細，可以在 [GitHub Copilot 訂閱管理頁面](https://github.com/settings/copilot) 或 [GitHub 支付與帳單說明](https://docs.github.com/en/billing/managing-billing-for-github-copilot/about-billing-for-github-copilot) 查看。
